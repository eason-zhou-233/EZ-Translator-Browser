# EZ Translator 翻译神器

一个轻量级浏览器翻译插件，同时支持传统谷歌翻译服务和基于大语言模型（LLM）的智能翻译。

---

## 功能特点

### 网页翻译

支持：

* 整页翻译
* 页面划词翻译
* 一键恢复原文
* 原地替换网页内容

自动跳过以下内容：

* URL 链接
* 邮箱地址
* 数字
* 日期
* 已经是中文的文本

---

### 自由翻译窗口

点击浏览器工具栏中的插件图标即可打开自由翻译窗口。

支持：

* 任意文本翻译
* 自动识别源语言
* 自定义源语言和目标语言
* 一键复制翻译结果

---

## 双翻译引擎架构

### 传统翻译服务

当前支持：

* Google Translate

无需配置 API Key，即可直接使用。

---

### 大模型翻译

支持任意符合 OpenAI 标准接口的模型服务。

例如：

* OpenAI
* DeepSeek
* OpenRouter
* SiliconFlow
* 火山方舟
* LM Studio
* Ollama
* 其他 OpenAI Compatible API

用户可自行配置：

* API 地址
* 模型名称
* API Key
* 采样参数

---

## 智能缓存

插件内置本地翻译缓存机制：

* 自动缓存翻译结果
* 减少重复请求
* 提高翻译速度
* 自动清理过期缓存

所有缓存均保存在浏览器本地。

---

## 支持语言

支持自动识别：

* 中文
* English
* Français
* Deutsch
* 日本語
* 한국어
* Русский

并支持在这些语言之间互相翻译。

## 安装方式

### 开发者模式安装

1. 下载或克隆本项目

2. 打开 Chrome：

chrome://extensions

3. 开启开发者模式

4. 点击：

加载已解压的扩展程序

5. 选择本项目目录

---

## 隐私说明

EZ Translator 不会收集、存储、上传或出售用户数据。

翻译请求将直接从浏览器发送到用户选择的翻译服务：

* Google Translate
* 用户自行配置的大模型接口

开发者无法访问用户的翻译内容。

详见：

PRIVACY_POLICY.md
